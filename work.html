Ecoactuar

Descripcion: Ecoactuar es una aplicacion web en la que puedes reportar problemas ambientales en tu ciudad ya sean desechos toxicos, maltrato animal, contaminacion del aire y el agua, entre otros. Este proyecto fue el trabajo final de un programa piloto que enseña a jovenes sobre programacion web el cual se llama Coderise.

Tecnologias Utilizadas: Para este proyecto se trabajaron con las siguientes tecnologias Node.js - Javascript - HTML5 - CSS3 - Bootstrap - Jquery
